Multi-PDF RAG Assistant v1

A Retrieval-Augmented Generation (RAG) assistant that allows users to ask questions across multiple PDF documents.

 Tech Stack

1 Google Colab (CPU)
2 LangChain
3 PyPDF
4 HuggingFace Embeddings
5 FAISS
6 Sentence Transformers
7 Transformers
8 TinyLlama

 Workflow

Multiple PDFs
↓
Load PDFs
↓
Chunking
↓
Embeddings
↓
FAISS
↓
Retriever
↓
TinyLlama
↓
Q&A

## Features

* Supports multiple PDF documents
* Semantic search with FAISS
* Context-aware question answering
* Interactive Q&A loop
* Built using open-source models and libraries
